SIH-ML-Chatbot
ML-based chatbot for Smart India Hackathon (SIH) preparation using TF-IDF and Cosine Similarity designed for students prepare for SIH.
Features:-
  1. ML-based response system (TF-IDF + Cosine Similarity)
  2. SIH-focused dataset
  3.Simple interactive chatbot UI
  4.Flask backend
Technologies Used:-
  1. Python
  2.Flask
  3.Scikit-Learn
  4.HTML, CSS, JavaScript
How to Run the Project:-
 1. Clone the repository:
   git clone https://github.com/yourusername/SIH-ML-Chatbot.git
 2. Navigate to project folder:
   cd SIH-ML-Chatbot
 3. Install required packages:
   pip install -r requirements.txt
 4. Run the application:
   python app.py
 5. Open in browser:
   http://127.0.0.1:5000/
Project Type:-
   Mini ML Project (NLP-based Chatbot)
